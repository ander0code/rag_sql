# RAG-SQL: Conversión de Lenguaje Natural a Consultas SQL

Guía Rápida de Instalación y Uso de RAG-SQL
Configuración del entorno

# Crear y activar un entorno virtual con Python 12+ (o 3.11+)

# Para Windows
python -m venv venv
.\venv\Scripts\activate

# Para macOS/Linux
python3 -m venv venv
source venv/bin/activate

# Instalar dependencias
pip install -r requirements.txt


# Ejecución del sistema
Una vez configurado todo, simplemente ejecuta:

python main.py -q "tu consulta en lenguaje natural"
